Puzzap is a user friendly application that can download zip and rar storage.
---
In the first use, you need to show Puzzap itself by saying "Choose another application" from the "Open With" menu in order to provide the file with ZIP or RAR extension.
---
Run a zip or rar file together. It will extract the contents to the location found. If it doesn't work; In the Registry Editor, in the "ROOT" section, under the "*" heading, create a key named "puzzap" and enter the location of "puzzap.exe" to its default value.
